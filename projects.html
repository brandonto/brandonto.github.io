Space Shooter
Aug 2014 - Feb 2015

A 2D scrolling space shooter written in C++ using SDL2. Based on old school arcade titles like Galaga and R-Type. This game was made to be as "ship-worthy" as possible. Which means a lot of time was put into the minor details.

Environment Simulated Study Room
Oct 2014 - Dec 2014

A projector system created for the Discovery Centre at Carleton University. The system provides synchronized video and audio streaming to three projectors. It can be controlled through the web application or an IR remote.
